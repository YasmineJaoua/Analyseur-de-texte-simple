# Analyseur-de-texte-simple
Lire  un texte et afficher : le nombre de mots, de lettres, d’espaces, de majuscules, de minuscules. Trouver le mot le plus long. Calculer la longueur moyenne des mots Calculer la fréquence de chaque lettre (ex. histogramme des lettres). Sauvegarder les résultats dans un fichier .txt.
